# Lessons

- Use plan mode for non-trivial tasks.
- Keep the first implementation minimal.
- Do not add advanced abstractions before the base pipeline works.
- With Prisma + Supabase, use DATABASE_URL for pooled runtime and DIRECT_URL for migrations.
- Do not run npm install inside a folder until package.json exists.
- Never paste real API keys or database passwords into code or committed files.
- Use `.env` for local secrets and `.env.example` for placeholders.
